//////////// PRUEBAS DE RENDIMIENTO ////////////////

- Para correr las pruebas de rendimiento se necesita el software J-meter Version 5.4.1
- Se debe copiar la libreria que se adjunta en este repositorio (jmeter-plugins-manager-1.6) en la carpeta "\\apache-jmeter-5.4.1\lib\ext"
- Una vez se inicie el software se debe ir al menu opciones, en la opcion "plugins manager", se desplegara una ventana en la cual se debe seleccionar
  "jpgc - Standart Set"
- Luego se abre el archivo "PruebasRendimiento.jmx" y se da clic en arrancar para empezar las pruebas.
